# Agente de IA - Premier League

Projeto desenvolvido para a disciplina de Inteligência Artificial.

O sistema funciona como um agente especializado em jogadores da Premier League, permitindo que o usuário faça perguntas em linguagem natural sobre estatísticas dos jogadores.


---

#  Funcionalidades

- Consultar estatísticas de jogadores
- Comparar jogadores
- Histórico de conversa
- Interface web com Streamlit
- Integração com IA local usando Ollama + Llama 3

---

#  Tecnologias utilizadas

- Python
- Streamlit
- Pandas
- Requests
- Ollama

---

#  Dataset

O projeto utiliza um dataset CSV contendo estatísticas de jogadores da Premier League.

Principais dados:
- gols
- assistências
- passes
- desarmes 
- cartões
- posição
- nacionalidade
- clube
